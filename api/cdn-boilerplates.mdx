The public Pxxl repository includes deploy-ready boilerplates under `public/boilerplates`.

| Boilerplate | Use case |
| --- | --- |
| `static-cdn-gallery` | A fast static gallery that renders CDN-hosted assets. |
| `vite-react-pxxl` | A React app configured for Pxxl deployment and CDN asset usage. |
| `express-api-pxxl` | A small API backend that can safely proxy CDN uploads with server-held API keys. |

Each boilerplate includes a `pxxl.boilerplate.json` manifest and an SEO-focused README that explains how to deploy the project on Pxxl.
