# How to Deploy a Next.js App in Nigeria — Free Hosting in 5 Minutes

Next.js is the most popular React framework for Nigerian developers building full-stack web applications. This guide shows you how to deploy a Next.js app — including API routes, SSR pages, and environment variables — for free in under five minutes.

## Prerequisites and Project Setup

Make sure your Next.js project is pushed to a GitHub or GitLab repository. Confirm your package.json has a 'build' script ('next build') and a 'start' script ('next start'). If you use environment variables, collect all of them — Pxxl will ask for them during setup. API routes, dynamic routes, middleware, and Image Optimization all work on Pxxl without additional configuration.

## Deploying Your Next.js App on Pxxl

Sign in to pxxl.app and click 'New Project'. Select your Next.js repository — Pxxl detects the framework automatically and sets the build command to 'npm run build' and start command to 'npm run start'. Add your environment variables in the Environment Variables section. Click Deploy. Pxxl installs your npm packages, runs the build, and starts the server. Your app is live at a \*.pxxl.app URL in under two minutes.

## Adding a Custom Domain and HTTPS

From your Pxxl project settings, click 'Domains' and enter your custom domain. Pxxl generates two DNS records — an A record and a CNAME — that you add at your domain registrar. After DNS propagation (usually 15-60 minutes), Pxxl automatically provisions a free Let's Encrypt SSL certificate. Your Nigerian users can now access your Next.js app at your own domain with full HTTPS.
