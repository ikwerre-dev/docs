# After buying a domain on Pxxl

Finish registration, DNS, project connection, proxy resync, and SSL coverage.

After payment is confirmed, Pxxl starts the registration and DNS setup flow. Once the domain appears in your dashboard:

## [Setup Steps](https://pxxl.app/#setup-steps)

1. Open the domain detail page.
2. Confirm DNS is managed by Pxxl.
3. Connect the domain to a project.
4. Click **Resync Proxy**.
5. Open the **SSL** tab and check coverage.

## [Pending Registration](https://pxxl.app/#pending-registration)

If registration is still pending, wait for the domain task to finish before expecting DNS or SSL to work.
